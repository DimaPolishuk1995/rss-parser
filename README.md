# Rss Parser News on ANGULAR

Requirement

  - node.js
  - grunt
  - bower

### Installation

```sh
$ git clone https://github.com/DimaPolishuk1995/rssParser.git
```

### Building

```sh
$ cd project
$ start dev.sh or ./dev.sh
```
