# Rss Parser News on ANGULAR

Requirement

  - node.js
  - gulp
  - bower
  - install CORS plugin

### Installation

```sh
$ git clone https://github.com/DimaPolishuk1995/rssParser.git
```

### Building

```sh
$ cd project
$ npm i and bower i
$ gulp serve or gulp build
```
